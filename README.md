# 🎙️ دليل تثبيت وتشغيل بوت الغرف المؤقتة

---

## 📋 المتطلبات

- Python 3.10 أو أحدث
- حساب على Discord Developer Portal

---

## 🔧 الخطوة 1: إنشاء البوت على ديسكورد

1. اذهب إلى: https://discord.com/developers/applications
2. اضغط **New Application** واختر اسماً للبوت
3. من القائمة الجانبية اضغط **Bot**
4. اضغط **Add Bot** ثم **Yes, do it!**
5. من نفس الصفحة اضغط **Reset Token** وانسخ التوكن
6. فعّل الخيارات التالية تحت **Privileged Gateway Intents**:
   - ✅ PRESENCE INTENT
   - ✅ SERVER MEMBERS INTENT
   - ✅ MESSAGE CONTENT INTENT
7. احفظ التغييرات

---

## 🔗 الخطوة 2: دعوة البوت للسيرفر

1. من القائمة الجانبية اضغط **OAuth2 → URL Generator**
2. في **SCOPES** اختر: `bot`
3. في **BOT PERMISSIONS** اختر:
   - ✅ Manage Channels
   - ✅ Move Members
   - ✅ Mute Members
   - ✅ View Channels
   - ✅ Connect
   - ✅ Send Messages
   - ✅ Embed Links
4. انسخ الرابط المولّد وافتحه لدعوة البوت

---

## 💻 الخطوة 3: تثبيت المكتبات

افتح Terminal أو Command Prompt وشغّل:

```bash
pip install -r requirements.txt
```

---

## ⚙️ الخطوة 4: إعداد التوكن

افتح ملف `bot.py` وغيّر السطر التالي:

```python
TOKEN = "YOUR_BOT_TOKEN_HERE"
```

ضع التوكن الذي نسخته بين علامتي الاقتباس.

---

## ▶️ الخطوة 5: تشغيل البوت

```bash
python bot.py
```

إذا ظهر:
```
✅ البوت شغال: [اسم البوت]
```
فالبوت يعمل بنجاح! 🎉

---

## 🏗️ الخطوة 6: إعداد السيرفر

في أي قناة نصية اكتب:
```
.v setup
```

سيقوم البوت تلقائياً بإنشاء:
- 📁 كاتيغوري: **🎙️ الغرف المؤقتة**
- 🎙️ قناة: **➕ إنشاء روم** (الأعضاء يدخلونها لإنشاء غرفتهم)
- 🚫 قناة: **🚫 المطرودون** (يُنقل إليها المطرودون)

> **ملاحظة:** البوت يُعدّ السيرفر تلقائياً حين يُضاف إليه، لكن `.v setup` يمكنك استخدامه في أي وقت لإعادة الإعداد.

---

## 📌 قائمة الأوامر

| الأمر | الوظيفة |
|-------|---------|
| `.v lock` | قفل الغرفة |
| `.v unlock` | فتح الغرفة |
| `.v hide` | إخفاء الغرفة |
| `.v unhide` | إظهار الغرفة |
| `.v name [اسم]` | تغيير اسم الغرفة |
| `.v limit [عدد]` | تحديد عدد الأشخاص (0 = غير محدود) |
| `.v bitrate [رقم]` | تغيير جودة الصوت بـ kbps |
| `.v reject @user` | طرد عضو إلى قناة المطرودين |
| `.v ban @user` | حظر عضو من الدخول نهائياً |
| `.v permit @user` | السماح لعضو محظور أو خاص |
| `.v claim` | أخذ ملكية الغرفة إن غاب المالك |
| `.v transfer @user` | نقل الملكية لشخص آخر |
| `.v help` | عرض قائمة الأوامر |

## ❓ مشاكل شائعة

**البوت لا يرد على الأوامر:**
- تأكد أن MESSAGE CONTENT INTENT مفعّل
- تأكد أن البوت لديه صلاحية قراءة الرسائل في القناة

**البوت لا يحرك الأعضاء:**
- تأكد أن البوت لديه صلاحية Move Members
- تأكد أن رتبة البوت أعلى من رتبة العضو

**الغرف لا تُحذف:**
- تأكد أن البوت لديه صلاحية Manage Channels
